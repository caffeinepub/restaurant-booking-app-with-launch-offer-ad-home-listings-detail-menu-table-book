# Building Android APK for TableBook

This guide provides step-by-step instructions to build a debug APK of the TableBook web app wrapped in an Android WebView shell.

## Prerequisites

1. **Android Studio** (latest stable version recommended)
   - Download from: https://developer.android.com/studio
   - Includes Android SDK and required build tools

2. **Java Development Kit (JDK)**
   - JDK 17 or higher required
   - Verify installation: `java -version`

3. **Node.js and pnpm**
   - Node.js 18+ required
   - pnpm installed globally: `npm install -g pnpm`
